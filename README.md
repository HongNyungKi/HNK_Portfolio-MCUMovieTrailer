# MCU Movie Trailer
----------

## 프로젝트 설명
마블 영화의 예고편을 세계관 순서대로 나열한 웹페이지입니다.

## 사용 스택
: HTML / CSS / JAVASCRIPT

## 프로젝트 특징
1. **json 데이터**를 서버에서 전달받았다고 가정하고, 이를 활용해 html을 구성하였습니다.
2. class를 활용하여 javascript를 **모듈화 구성**하였습니다.
* 반응형을 고려하지 않았습니다.
* ajax통신이 아닌 XMLHttpRequest객체를 활용하여 데이터를 다루었습니다.
